# Zomato Data Analysis Project

## Project Overview
This project analyzes the Zomato restaurant dataset to understand factors affecting restaurant ratings and popularity.

## Objectives
- Analyze restaurant ratings
- Understand the effect of online ordering
- Explore the impact of cost on customer votes
- Identify important factors affecting high ratings

## Dataset Features
- online_order
- book_table
- rate
- votes
- approx_cost(for two people)
- listed_in(type)

## Key Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Regression Modeling

## Key Insights
- Restaurants with online ordering tend to receive more votes
- Dining restaurants receive higher engagement
- Cost has a moderate relationship with popularity

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Statsmodels

## Project File
Zomato.ipynb contains the full analysis and model building process.
